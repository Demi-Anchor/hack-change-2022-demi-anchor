#### Локальный запуск

1. Создать файл ".env" с необходимыми переменными окружения рядом с файлом "Makefile"

   *.env example*

   ```
   DB_HOST=dbhost.ru
   DB_PORT=5432
   DB_NAME=beautifulname
   DB_USER=thebestuser
   DB_PASSWORD=verystrongpassword
   ```

2.  Выполнить make run в консоли
